Resources

General Python Reference
  • Official Python Language website
  • Google Python Style Guide: invaluable resource on language and style rules
  • Python Books: tons of open-sourced books on Python
  • Hidden Features of Python: may be out of date, and definitely a next step, but may still be a worthwhile document for exploring Python obscurities
  Courseware
  • Coursera: university courses, some award certificates
  • Learn Python the Hard Way: 52 exercises to get you going with Python basics
  • Codecademy: tracks in Python, JavaScript, HTML, Ruby, etc.
  • Udacity: “Introduction to Computer Science” is a popular course, teaching how to build a search engine; there are other courses on offer as well
  • EdX: courses from MIT, Harvard, Berkeley, and the UT system
  • MIT OpenCourseware: variety of courses in Engineering
  • Learn Python: learn the basics thru advanced topics with this online tutorial
  • Google’s Python Class: part of Google’s Code University; video-based
  • “The Best Way to Learn Python”: A Nettuts guide with a list of great links to all kinds of different resources
  • Boston Python Workshop curriculum: same one used at Portland Python Workshop
  • ShowMeDo: ugly but honest edu-share site for FOSS community
  Frameworks
  • Django: most popular Python web framework
  • Official Django tutorial
  • Learn Django: tutorials and resources for Django newbies
  • Django Sites: look at source code from real Django sites
  • Test-Driven Development with Django tutorial
  • Getting Started with Django on Heroku
  • Django Best Practices: style tips, as well as how to build, deploy, and maintain a usable app
  • Kenneth Love’s “Getting Started with Django” series
  • Flask: a “microframework” for small Python apps
  • Official Flask tutorial
  • SQLAlchemy and You: You’ll probably want to read this article on SQLAlchemy before you try Flask
  • Pyramid: another popular framework
  • Tastypie: RESTful APIs for Django (not a standalone framework like the others, but rather a web service API framework for creating REST-style interfaces)
  • A good article on the use and importance of REST, for the unfamiliar
  Popular Python Modules
  • Useful Modules List: not exhaustive, but a list of useful modules geared towards beginners)
  • PyJamas: Comprehensive Widget toolkit, P-to-JSc compiler, and AJAX library
  • NumPy and SciPy: modules that aid in Math and Science applications; SciPy requires NumPy
  • PyPy: alternative compiler for CPU-intensive applications
  • BeautifulSoup: HTML/XML parser forscraping HTML pages
  • Python Image Library: aids in the processing and manipulation of images
  • Matplotlib: Plotting for 2D graphing and charting
  • PyGame: Game programming
  • Python and Cryptography
  ￼Getting Involved in OpenSource Mostly reference materials; projects in need of contribution will be added in an ad hoc fashion:
  • OpenHatch: First-stop resource for getting involved in FOSS as a beginner
  • Teaching Open Source: a wealth of information on teaching OpenSource, with some additional information on wading into the FOSS world
  • PyLadies LA’s Barbara Shaurette blogs: “Open Source Contributions Come in All Shapes and Sizes” with links to get involved)
  • 14 Ways to Get Involved in OpenSource...: advice for newer programmers on starting in tickets, bug fixes, documentation, etc.
  • Django Book: online Django tutorial needs help updating; writers/editors and experienced Django users apply
  • Getting Involved....with the Django project itself
  • Slides from the 2012 Open Source Conference, held in Portland; some inspiration, if nothing else!
  • Google’s open source portal
  Git/GitHub/SourceForge
  • GitHub help: installing git, opening a free GitHub account, choose a client, and learn about how GitHub works
  • Try Git: git/version control basics with a browser-based mock command-line interface; offered through Code School
  • “Pam’s Intro to Git with Github”: tutorial focuses on how to use Github to get started with Free and Open Source
  • Git Ready: Beginner--through--Advanced git tutorials, plus resource list
  • Github's online courses: free online classes offered through GitHub
  • SourceForge: like GitHub in some ways; lists projects that need help and can be downloaded
  • BitBucket: is much like Github, but offering free private repos as well...
  Get Help
  • StackOverflow: ask, and you shall receive....answers. And probably some attitude.
